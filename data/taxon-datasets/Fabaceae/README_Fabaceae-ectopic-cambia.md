# Fabaceae Ectopic Cambia Dataset

This taxon-specific dataset documents the occurrence and structural diversity of ectopic cambia in Fabaceae.

The dataset is maintained in a dedicated repository to support independent development, collaborative authorship, and versioned releases.

GitHub repository  
https://github.com/ilcneto/fabaceae-ectopic-cambia-dataset

Zenodo archive  
DOI: https://doi.org/10.5281/zenodo.19103807

This dataset is part of the **Plant Vascular Variants Database**, which integrates global and taxon-specific datasets documenting vascular variants across seed plants.
