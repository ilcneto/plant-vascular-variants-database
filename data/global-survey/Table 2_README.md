# Distribution of vascular variants across seed plants.											This table lists vascular variants across taxa.																						## Columns											- ### Clade 		Contains the plant class (e.g., gymnosperm or angiosperm).	- ### Order 	Indicate the plant order.  	- ### Family 	Indicate the plant family.	- ### Vascular Variant Category 	Indicate the vascular variant category.	- ### Vascular Variant Pattern	Indicate the vascular variant pattern.	- ### Morphotype	Indicate the subtype for the given pattern (if any).	- ### References	List references that describe the ontogeny and anatomical description for the vascular variant pattern.
